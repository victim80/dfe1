DumpChk

Source: https://docs.microsoft.com

DumpChk (Microsoftâ€™s crash dump file checker tool) is a program that performs a quick analysis of a crash dump file. It shows summary information about what the dump file contains. If the dump file is corrupt in such a way that it cannot be opened by a debugger, DumpChk reveals the same to the investigator.

Syntax:

DumpChk [-y SymbolPath] DumpFile

Parameters:

  • -y SymbolPath: SymbolPath specifies where DumpChk needs to search for symbols
  • DumpFile: DumpFile specifies the crash dump file that is to be analyzed
